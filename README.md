# HotShare
### How to use SharpAI app
#### 1. Download sharpAI app called '来了吗' from Appstore or Android app market(search lailema)
![install](https://github.com/SharpAI/mobile_app_server/blob/android_porting/app_intro/appstore.png)

#### 2. Sign up
![sign_up](https://github.com/SharpAI/mobile_app_server/blob/android_porting/app_intro/login.png) 

#### 3. Create your own group 
![create](https://github.com/SharpAI/mobile_app_server/blob/android_porting/app_intro/home.png)
![create2](https://github.com/SharpAI/mobile_app_server/blob/android_porting/app_intro/init_group.png)

#### 4. Scan your device and link it to your group
![scan](https://github.com/SharpAI/mobile_app_server/blob/android_porting/app_intro/scan_box.png)
![link](https://github.com/SharpAI/mobile_app_server/blob/android_porting/app_intro/link_group.png)

#### 5. Go to msg page
![gotomsg](https://github.com/SharpAI/mobile_app_server/blob/android_porting/app_intro/gotomsg.png)
![msg](https://github.com/SharpAI/mobile_app_server/blob/android_porting/app_intro/msg%20page.png)

#### 6. Go to labelling page and label new face
![gotolabel](https://github.com/SharpAI/mobile_app_server/blob/android_porting/app_intro/gotolabel.png)
![choose_img](https://github.com/SharpAI/mobile_app_server/blob/android_porting/app_intro/choose_img.png)
![label_name](https://github.com/SharpAI/mobile_app_server/blob/android_porting/app_intro/label_name.png)
